# Promoter Sequence Generation Plan (gen-promoter)

## Goal
- gen-promoter는 `gen-seq-and-emb.ipynb`와 다른 입력/처리 흐름을 전제로,
  입력 promoter 구간을 기존 모델 입력 포맷으로 맞추고 서열 생성까지만 수행한다.
- 초기 단계는 gene 1개(HBB)만 대상으로 정상 동작을 확인한다.

## Steps
1. **gen-promoter 입력 스펙 정리**: 현재 노트북에서 promoter를 어떤 방식으로 받는지(좌표/파일/직접 입력) 명시한다.
2. **HBB 단일 타깃 흐름 확정**: HBB에 대해 필요한 메타데이터(종/버전/좌표 등)와 입력 값을 고정한다.
3. **promoter 처리 단계 정의**:
   - 좌표 입력: 참조 게놈 로딩 경로와 추출 함수 연결.
   - 서열 입력: 길이/알파벳/strand 정합성 검사.
4. **모델 입력 포맷 정규화**: 기존 모델이 기대하는 입력 포맷(길이/토큰/패딩/특수문자)을 맞춘다.
5. **서열 생성 연결**: 정규화된 입력을 사용해 sequence generation 함수/클래스로 연결한다.
6. **출력 최소화**: 임베딩은 제외하고 생성된 서열만 저장/표시한다.
7. **노트북 셀 구성**: `gen-promoter.ipynb`에 입력 → 처리 → 정규화 → 생성 → 출력 순으로 셀 구조를 설계한다.
